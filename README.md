# 反编译-bcm黑客三件套

本仓库仅用于安全研究、逆向工程学习、恶意样本识别与防护教育。

## 用途声明

- 仅限学习、教学、研究和防御性分析。
- 不得用于攻击、入侵、盗号、爆破、DDoS、破坏服务、绕过风控或其他违法违规行为。
- 不得直接运行样本，除非在隔离的虚拟机、沙箱或受控实验环境中进行。
- 不得将样本、分析结论或提取出的代码用于二次分发、商业利用或生成可执行攻击工具。

## 仓库内容

- `file/`：待分析的 Windows PE/Tauri 样本。
- `reverse_output/`：静态逆向输出、提取出的前端资源和分析报告。

## 安全提醒

这些文件可能具备下载远端程序、写入临时目录、执行外部 EXE 等风险行为。不要在日常使用的主机上运行。需要动态分析时，请使用无敏感账号、无共享目录、网络受控的隔离环境。
