# 反编译-bcm黑客三件套

本仓库仅用于安全研究、逆向工程学习、恶意样本识别与防护教育。

## 用途声明

- 仅限学习、教学、研究和防御性分析。
- 不得用于攻击、入侵、盗号、爆破、DDoS、破坏服务、绕过风控或其他违法违规行为。
- 不得直接运行样本，除非在隔离的虚拟机、沙箱或受控实验环境中进行。
- 不得将样本、分析结论或提取出的代码用于二次分发、商业利用或生成可执行攻击工具。

## 仓库内容

- `file/`：待分析的 Windows PE/Tauri 样本。
- `reverse_output/`：静态逆向输出、提取出的前端资源和分析报告。
- `sandbox_test_report/`：云沙箱样本分析报告，仅供交叉参考。

## 安全提醒

这些文件可能具备下载远端程序、写入临时目录、执行外部 EXE 等风险行为。不要在日常使用的主机上运行。需要动态分析时，请使用无敏感账号、无共享目录、网络受控的隔离环境。

## 云沙箱报告说明

`sandbox_test_report/` 中包含对三个样本的微步在线云沙箱报告。报告中的判定结果均为“未知”，这不代表样本无害，也不代表不存在网络连接或本地风险行为。

云沙箱结果可能受运行时间、网络环境、权限、交互触发条件、远端地址可用性、反沙箱逻辑等因素影响。当前仓库中的静态逆向结果已经显示样本包含可疑网络 URL、管理员权限检查、写入 `C:/Windows/Temp/.../install.exe`、以及执行外部程序的相关行为。因此这些云沙箱报告只能作为参考，不应作为安全放行依据。
